## 任务
基于“本轮对话中已经完成的任务与结果”，生成一段可复用的 Markdown 记忆块，方便用户手动保存到仓库。

## 要求
- 不要向用户提问
- 不要解释过程
- 只输出一段 Markdown（从 `---` 开始到结尾）
- 内容必须足够让下次类似任务时，agent 先读它就能复用做法
- 用中文
- 尽量具体：包含命令、关键路径、决策规则、踩坑点
- 若本轮缺少某些信息（例如具体文件名/命令输出），不要编；用 `（待补充：...）` 标注
